# API de Tarefas (.NET)

API simples desenvolvida em C# utilizando ASP.NET Core para gerenciamento básico de tarefas.

## Funcionalidades

- Listar tarefas
- Adicionar novas tarefas

## Tecnologias

- C#
- ASP.NET Core
- Web API

## Objetivo

Projeto criado para praticar conceitos básicos de desenvolvimento de APIs REST com .NET.
